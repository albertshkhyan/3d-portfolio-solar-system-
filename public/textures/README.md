# Planet textures

Place the following image files in this folder. They are used for the 3D solar system and mobile nav.

**Source:** [Solar System Scope – Free textures](https://www.solarsystemscope.com/textures/)

Download the **2k** versions and use these exact filenames:

| File | Used for |
|------|----------|
| `2k_sun.jpg` | Sun |
| `2k_mercury.jpg` | Mercury (About) |
| `2k_venus_surface.jpg` | Venus (Skills) |
| `2k_earth_daymap.jpg` | Earth (Experience) |
| `2k_mars.jpg` | Mars (Projects) |
| `2k_jupiter.jpg` | Jupiter (Architecture) |
| `2k_saturn.jpg` | Saturn (DevOps) |
| `2k_saturn_ring_alpha.png` | Saturn rings |
| `2k_neptune.jpg` | Neptune (Contact) |
| `2k_moon.jpg` | Project moons |
| `2k_stars_milky_way.jpg` | Starfield background |

Optional: `2k_earth_clouds.jpg` (Earth clouds overlay).

If a file is missing, the app shows a solid color fallback for that body.
